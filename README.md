# Prediction of MLB Player Performance for the 2025 Season

## Created by Marcus Saffold and Jack Fritschel for STAT 4050: Intro to Stat Computing

- Used statistical models to determine if we can predict an MLB player's postseason performance based off their regular season performance in the 2025 season
- Created, cleaned, and analyzed large MLB datasets
- Used a scatterplot, correlation matrix, linear regression and random forest to solve our research questions
- Presented to the UMD math and stats department on 12/4/25
- Open MLB_Presentation.pdf to view summary slides of project
- Download MLB_Predictor_Models.R and its respective .csv files to run source code for yourself
